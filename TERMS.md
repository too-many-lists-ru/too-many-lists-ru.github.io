# Перевод терминов

Чтобы не сбивать читателя с толку, технические термины в книге переводятся согласованно.
Разные английские термины имеют разный русский перевод.
Скажем, для row, line и string нужно подыскать разные варинаты перевода.

Термин должен всегда переводиться однообразно, не смотря на то, что разные книги части переводят разные переводчики.

В файле TERMS собраны термины, которые мы используем.

---------------------------------------------------------------------
| aliasing             | псевдонимы                   | |

| amortization         | амортизированная сложность   | |
| borrowing            | заимствование                | |
| cache                | кэш                          | |
| desctructor          | деструктор                   | |
| deque                | дек                          | |
| double linked        | двухсвязный                  | |
| generics             | обобщения                    | |
| inherited mutability | наследуемая мутабельность    | |
| interior mutability  | внутренняя мутабельность     | |
| intrusive list       | интрузивный список           | |
| ownership            | владение                     | |
| pattern matching     | сопоставление с образцом     | |
| persistent           | устойчивый                   | |
| raw pointers         | сырые указатели              | |
| single linked        | односвязный                  | |
| stacked borrow       | многоуровневое заимствование | |
| toolchain            | инструментарий               | |
| variance             | вариантность                 | |
